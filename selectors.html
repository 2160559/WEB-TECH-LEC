CSS Selectors Level 3

Selectors are patterns(2) that are used in CSS to select an element(1) in an xml or html file and specify a style for the selected elements.
1. An element usually consist of a start tag , a content and an end tag.
Example: (to label the parts)
<title> Web System and Technologies </title>
<p> Enter a paragraph </p>

2. Selectors follow the pattern :
One or more sequence of simple selectors separated by combinators and an optional pseudo-element that may be appended to the last sequence of the simple selector
  • A sequence of simple selectors is a chain of simple selectors that are not separated by a combinator. It always begins with a type selector or a universal selector. No other type selector or universal selector is allowed in the sequence. (W3C Candidate Recommendation)
    • A simple selector is either a type selector, universal selector, attribute selector, class selector, ID selector, or pseudo-class.

  • The following may be used as a combinator
    Whitespace, > , +, ~
    • Whitespace: Only the characters "space" (U+0020), "tab" (U+0009), "line feed" (U+000A), "carriage return" (U+000D), and "form feed" (U+000C) can occur in whitespace. Other space-like characters, such as "em-space" (U+2003) and "ideographic space" (U+3000), are never part of whitespace.
  • Pseudo elements are used to select contents of an element.

Invalid Selector
An empty selector, containing no sequence of simple selectors and no pseudo-element

Simple Selectors used to define the element(s) to be styled.

Type Selector
Use: Selects all the elements that match the specified element name.
Syntax: p
Example: p will match any <p> element.
Class Selector
Use: Selects all the elements that have the specified class attribute preceded by “.”
Syntax: .classname
Example: .classname will match all elements that have a class name, “classname”
ID Selector
Use: Selects an element that has the specified id attribute preceded by “#”
Syntax: #idname
Example: #idname will match all elements that have a class name, “idname”
Universal Selector
Use: Selects all elements.
Syntax: *
Example: div * selects all elements within the
Attribute Selector
Use: Selects all elements matching the given attribute name
Syntax: a[attributename]
Example: a[attributename] selects an element with an attribute name attributename
Pseudo-Classes
a. Dynamic Pseudo-Classes – classify elements on characteristics other than their name, attributes, or content.
i. Link Pseudo-Classes – displays links depending on whether it is visited or not.
ii. User Action Pseudo-Classes – changes rendering in response to user actions
b. Target Pseudo-Classes – selects an element with anchor identifier
c. Language Pseudo-Classes – selects an element with a specified language
Syntax: :lang([language])
Example: :lang(en) selects a n element with language en(English).
d. UI Element States Pseudo-Classes – selects an element depending on the state of that element
e. Structural Pseudo-Classes – selects elements based on extra information that cannot be represented by other simple selectors or combinators
