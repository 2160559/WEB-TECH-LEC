Cascading Style Sheet (CSS)

Can be used for all structural markups, mainly used for the aesthetics of a page. Adapted in HTML v. 4.0.

Types of CSS

  1. Internal 

    <html>
	<style>
		body {
			background-color: green;
		}
	</style>
</html>

  2. External (recommended because of reusability and consistency)

    <link rel="stylesheet" href="style.css">

  3. Inline

    4. <h1 style= “color:blue; text-align:center”> webtech </h1>
