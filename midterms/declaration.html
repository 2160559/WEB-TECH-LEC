CSS Declaration

Properties

These allows authors to specify the values of several properties with a single property. Examples include background, margin, border, padding and font.

Vendor Specific/Vendor Prefixes

These are used by browser vendors as a prefix for the names of experimental or non-standard CSS properties. Lately, however, vendors are moving away from vendor prefixes in favor of user-controlled flags or preferences. Examples include -webkit-, -moz-, -o-, and -ms-.
