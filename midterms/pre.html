CSS Preprocessors, Frameworks and Polyfills

Preprocessors

These generate CSS using a custom language syntax that typically includes features that donâ€™t normally exist in pure CSS.

  • Examples: Sass, Less, and Stylus

Frameworks

These provide predefined CSS design functionality that can be reused, extended or customized.

  • Examples: Bootstrap, Foundation, and Materialize

Polyfills

These provide features that developers expect browsers to provide natively.

  • Examples: Modernizr and Selectivizr
