What is the World Wide Web?

The World Wide Web is a collection of resources and applications used to access resources via the internet. It was invented by Tim Berners-Lee in 1989, for the purpose of sharing resources, such as researches, among scientists.

Web Resources can be in static or dynamic in form. Static resources exist in the form that we access from the web server, such as HTML files that lies within the web. Dynamic resources are generated on the fly, and may vary differently per user when they access the resource. An example would be PHP scripts.

Web applications are applications used to access the resources from the web via the internet. There are two classifications of web applications:

  1. Client Applications
    • These are applications that are used by clients to access resources from the web.
    • Examples of client applications include browsers and apps.
  2. Server applications
    • These are applications that host dynamic applications.
    • Examples of server applications include Apache Web Server, Tomcat and IIS.
